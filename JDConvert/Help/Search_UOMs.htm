Summary

(NOT YET SUPPORTED)

This mode allows you to enter a text value along with UOM name or suffix to automatically detect the UOM entered.

 

Enter Search Criteria

At the top of this screen, enter a value to search. Be sure to include either the UOM's name or suffix after the value. Note that suffixes are case-sensitive.

  • Either press "Enter" on the keyboard, or press the magnifier button to the right, to perform the search.
  • If found, the list below will be populated with all possible converted values.
  • If not found, an error message will be displayed.

 

Equivalents

Upon searching at the top, the main list below will automatically refresh to show all possible UOM conversions.

This list contains a number of columns:

  • Equivalent Value - The converted value of each possible UOM.
  • Unit of Measure - The name of the UOM (automatically uses singular or plural depending on value).
  • Suffix - The suffix of the UOM (case-sensitive, also a unique identifier for the UOM).
  • Systems - The measurement systems, separated by commas, related to the UOM.

You can double-click any value in the list to copy it to the clipboard as text, and paste it elsewhere. This copies the Value and the Suffix, separated with a space.

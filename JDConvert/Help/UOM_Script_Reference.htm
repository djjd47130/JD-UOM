Summary

Below are all the possible capabilities of the UOM scripting feature.

Enumerations

  • TUOMType - Defines the overall type of UOM - used by TUOM object.
    • uomFactor
    • uomFormula
    • uomMetric
    • uomCombo
  • TUOMMetricUnit - Defines a specific unit of the Metric system.
    • msYocto
    • msZepto
    • msAtto
    • msFemto
    • msPico
    • msNano
    • msMicro
    • msMilli
    • msCenti
    • msDeci
    • msBase
    • msDeca
    • msHecto
    • msKilo
    • msMega
    • msGiga
    • msTera
    • msPeta
    • msExa
    • msZeta
    • msYotta
  • TUOMMetricUnits - Set of TUOMMetricUnit enum.

Records

  • TUOM - Carries the details of a particular Unit of Measurement.
    • UOMType: TUOMType;
    • ParentUOM: String;
    • Category: String;
    • NameSingular: String;
    • NamePlural: String;
    • Suffix: String;
    • Systems: String;
    • Factor: Float;
    • FromBase: String;
    • ToBase: String;
    • Aliases: String;
  • TUOMValue - Carries the details of a particular numeric value with a UOM attached to it.
    • UOM: String;
    • BaseValue: Float;
    • ConvertedValue: Float;

(NOT YET FULLY SUPPORTED)

This mode allows you to write ane execute complex scripts using UOMs.

Script Syntax

This uses Delphi Web Script, which allows you to write script using Pascal syntax - much like Delphi. Except it allows you to do much more.

The idea is that you can write a large script file and execute it to perform UOM conversions and calculations to your requirements.

Supported Syntax:

  • Basic symbols like ( ) ; : , ' etc..
  • Basic pascal functionality like for loops, constants, variables, functions and procedures.
  • Access to UOM library's functionality

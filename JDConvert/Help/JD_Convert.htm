Summary

This application allows you to convert units of measurement, as well as see various details / statistics of those UOMs. Units are separated into "Categories", such as "Distance", "Area", "Volume", "Mass", "Temperature", etc. Each category has a "Base" UOM assigned to it, typically the Metric system. For example, the base of "Distance" is "Meter", and the base of "Mass" is "Gram".

Main Menu

Along the left side of the screen is the Main Menu. These buttons allow you to navigate to the different sections of the application.

  • Normal Conversion  - Allows you to enter a numeric value and select a UOM to see all possible conversions to other UOMs.
  • Search UOMs  - Allows you to enter a text value along with UOM name or suffix to automatically detect the UOM entered.
  • UOM Details  - Provides detailed information about all registered UOMs along with a comparison chart.
  • UOM Builder  - Allows you to create your own custom UOMs.

 

System UOMs

Upon installation of JD Convert, the installer allows you to choose which UOM categories you want installed (Distance, Area, Temperature, etc.). These are referred to as the "System" UOMs, or otherwise "built-in", and cannot be directly modified.

 

Custom UOMs

JD Convert supports the ability to create and manage your own custom UOMs. You may create UOMs for an existing category (such as Distance), or your own entirely different category. Custom UOMs are available system-side along with the built-in ones.

See UOM Builder for more information.

 

UOM Names

Each UOM has both a singular and plural name assigned to it. These act as unique identifiers for the UOM - duplicates are not permitted.

  • Singular names are the MAIN unique identifier of the UOM.
  • Plural names are loosely treated as a secondary unique identifier of the UOM.

 

UOM Suffixes

Each UOM has a suffix assigned to it. This is the text that gets inserted after a value of that UOM. For example, 123.45 Kilometers would be 123.45 km. These act as a case-sensitive unique identifier for the UOM - duplicates are not permitted.

 

UOM Categories

Each UOM has a category assigned to it. This is the overall grouping of UOMs which can be converted between each other. For example, Distance, Area, Volume, Mass, Temperature, etc.

 

UOM Systems

Each UOM has one or more systems assigned to it. These are a larger grouping of conversion systems, such as Metric, Imperial, US Customary, etc.

Summary

This mode allows you to enter a numeric value and select a UOM to see all possible conversions to other UOMs.

 

 

Select UOM and Value

Use the controls at the top to select the desired unit of measurement and a value to be converted.

  • Category - Select the overall UOM category (Distance, Area, Volume, Mass, Temperature, etc.)
    • Changing this will refresh the "UOM" drop-down.
  • UOM - Select the specific UOM (Meter, Acre, Milliliter, Kilogram, Kelvin, etc.)
  • Value - Enter the numeric value you would like to convert.

 

Equivalents

Upon changing any of the controls at the top, the main list below will automatically refresh to show all possible UOM conversions.

This list contains a number of columns:

  • Equivalent Value - The converted value of each possible UOM.
  • Unit of Measure - The name of the UOM (automatically uses singular or plural depending on value).
  • Suffix - The suffix of the UOM (case-sensitive, also a unique identifier for the UOM).
  • Systems - The measurement systems, separated by commas, related to the UOM.

You can double-click any value in the list to copy it to the clipboard as text, and paste it elsewhere. This copies the Value and the Suffix, separated with a space.

Summary

JD Convert is written by Jerry Dodge, and is hosted free and open-source on GitHub:

djjd47130/JD-UOM: JD Library for Converting Various Units of Measurement. (github.com)

 

UOM Library

This application is only part of the UOM infrastructure. Behind the scenes is a Delphi library for managing UOM conversion.
